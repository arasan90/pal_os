Waits for one or more signals to be set.

Parameters
[in]signalSignal object to wait on.
[in]maskBitmask of signals to wait for.
[out]received_signalsPointer to store the bitmask of received signals.
[in]clear_maskIf 1, clear received signals before returning; if 0, do not clear the signals.
[in]wait_allIf 1, wait for all specified signals; if 0, wait for any.
[in]timeout_msTimeout in milliseconds. Use PAL_OS_NO_TIMEOUT for non-blocking or PAL_OS_INFINITE_TIMEOUT for infinite wait.
Returns
PAL_SIGNAL_SUCCESS if signals are received within the timeout, PAL_SIGNAL_TIMEOUT if timeout occurs, or PAL_SIGNAL_FAILURE on error.
Note
This function blocks until the specified signals are set or the timeout occurs.
