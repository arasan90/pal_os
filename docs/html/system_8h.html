pal_system_t * pal_system_get_stats ( void  )

Retrieves system statistics. This function provides access to system-heap statistics.

Returns
A pointer to a pal_system_t structure containing the system statistics. The caller should ensure proper handling of the returned pointer.
Note
Not all information are reliable on all platforms (e.g. on Linux it is not possible to have a valid heap size)

Parameters
